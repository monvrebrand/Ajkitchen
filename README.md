# AJ KITCHEN - Premium Ghanaian Cuisine

A premium food ordering platform for **AJ KITCHEN**, a homemade Ghanaian/African food business based in Columbus, Ohio.

## Features
- **Canvas Hero Scrollytelling**: An immersive 60fps image sequence animation showing deconstructed Ghanaian meals.
- **Pre-order System**: Weekly ordering windows with weekend delivery logistics.
- **Admin Kitchen Console**: Real-time order management, menu updating, and kitchen analytics.
- **Pink & White Aesthetic**: High-end minimalist design focused on professional food presentation.

## Tech Stack
- **Next.js 14** (App Router)
- **Supabase** (Auth & Database)
- **Tailwind CSS** (V4)
- **Framer Motion** (Scroll animations)
- **HTML5 Canvas** (High-performance rendering)

## Getting Started

1. Configure Supabase environment variables in `.env.local`.
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`
4. Access kitchen admin at `/admin/login`.

## Deployment
Optimized for Vercel deployment with edge-ready Supabase integration.
